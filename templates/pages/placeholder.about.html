About Page

This is a demo for Middle Seat created by Rina Schiller.

The events are taken from Ottawa Digital Facebook Events

This app uses Flask for the web framework, Facebook API, Mapbox JS SDK, and Heroku for web deployments

In order to see the events, the user must have the proper permissions on the Facebook page in accordance with the Facebook API Page access guidelines

In the future, a better solution would be to have a database that has all the events saved in it. The application could query the Facebook API a few times a day for updated events and put them into the database.
